## 🎯 문항 생성 계획

- 각 항목당 기본 1문항 (mcq 또는 diagnosis4)
- 중요 항목(입력검증/보안기능 핵심)은 2~3문항 (언어 변종)
- **예상 총 문항: 100~150개**

### 파일 네이밍 규칙

```
kisa-module/seed/design/
├── 01-input-validation.json        (DSG-IV-01 ~ DSG-IV-10)
├── 02-security-feature.json        (DSG-SF-01 ~ DSG-SF-08)
├── 03-error-handling.json          (DSG-EH-01)
└── 04-session-control.json         (DSG-SC-01)

kisa-module/seed/implementation/
├── 01-input-validation.json        (IMP-IV-01 ~ IMP-IV-17)
├── 02-security-feature.json        (IMP-SF-01 ~ IMP-SF-16)
├── 03-time-state.json              (IMP-TS-01 ~ IMP-TS-02)
├── 04-error-handling.json          (IMP-EH-01 ~ IMP-EH-03)
```
