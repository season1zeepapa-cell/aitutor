## 누락 기능 목록

### 1. 공통 (P0)

- 토스트 알림 시스템 (showToast)
- 이미지 확대 모달
- 에러 바운더리

### 2. 문제관리 탭 (P0)

- 문제 추가 모달 (번호, 본문, 선택지 4개, 정답, 해설, 카테고리/시험/과목)
- 문제 수정 모달
- 체크박스 전체선택 + 일괄 과목지정
- 일괄 삭제

### 3. DocStore 연동 탭 (P0)

- 카테고리 선택 + 시험이름 입력 (이관 대상 지정)
- 대상조회 체크박스 + 전체선택
- 문제이관 버튼 (선택/전체 이관)
- 이관완료 체크박스 + 전체선택
- LLM 프로바이더/모델 선택
- 해설생성 버튼 (선택/전체 해설 생성)
- 소스삭제 버튼
- 진행 상태 표시 바
- 처리 로그 (info/success/error + 복사/초기화)

### 4. 설정 탭

- 카테고리 수정 (이름, 정렬)
- 과목 수정/삭제
- 시험별 카테고리 지정
- 회원 권한 토글 (관리자 부여/해제)
- 회원 삭제

### 5. 문제풀이 탭

- 메모 첨부파일 (업로드/다운로드/삭제/미리보기)
- 문제 이미지 클릭 시 확대 모달
- 내용 복사 버튼
- 헤더 카테고리 선택 (전체 앱 범위 필터)
